CSS File and Structure

Pump HTML Template have are 5 css files:

1. Style.css ( Main CSS use for all pages )
2. Responsive.css ( Use for all pages responsive )
3. flexslider.css ( Use for slider ) 4. prettyPhoto.css
5. magnific-popup.css
6. font-awesome.min.css ( Use for all icons )

Here sample structure of CSS use on this theme

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

 

Javascript / Jquery

This theme imports 15 Javascript files, here the detail :

1. gauge.min.js
2. imagesloaded.pkgd.min.js
3. isotope.pkgd.min.js
4. jquery.easing.1.3.js
5. jquery.fitvids.js
6. jquery.flexslider-min.js
7. jquery.js ( Main Jquery File )
8. jquery.magnific-popup.min.js
9. jquery.prettyPhoto.js ( prettyPhoto is a jQuery lightbox clone. Not only does it support images, it also support for videos, flash, YouTube, iframes and ajax. )
10. jquery.smooth-scroll.js
11. jquery-ui.min.js
12. script.js
13. tinynav.min.js ( tinynav.js is a JavaScript plugin that lets you convert your website navigation into a select drop-down menu )
